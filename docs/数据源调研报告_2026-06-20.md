## 各源深度评估

### 1. 和风天气 (qweather) ⭐⭐⭐⭐⭐
**已有适配器**: `src/lib/api/qweather.ts`
- **免费版**: 1000次/天，非商业全免费（含灾害预警、AQI、台风、天文）
- **付费**: 按量计费，无预付
- **API v7.15**: 实时/24h逐小时/30天逐日/分钟降水/格点天气(3-5km分辨率)
- **中国覆盖**: 3000+市县区，格点全球坐标
- **优势**: 国内气象数据最全、含灾害预警、中文完美支持
- **劣势**: 免费额度1000次/天偏少
- **推荐**: **主力源 #1**，负责实时+预警+AQI

### 2. 高德地图 (amap) ⭐⭐⭐⭐ **角色变更：从天气兜底 → 空间数据基础设施**
**已有适配器**: `src/lib/api/amap.ts`
- **免费版**: **天气仅 5,000次/月**（⚠️ 之前"10万次/天"结论错误！），基础LBS 150,000次/月，基础搜索 5,000次/月
- **天气数据**: 实时天气 + 未来4天预报（温度/风力/湿度/天气描述），共计6字段，**无小时级、无降水量(mm)、无AQI**
- **空间数据**: 逆地理编码（GPS→adcode+区县名）、行政区查询（城市→所有区县+边界polyline）、坐标转换
- **中国覆盖**: 全国省市区县+边界坐标
- **Android SDK**: MapView底图 + Polygon绘制 + Marker覆盖 — 原生支持地图可视化
- **优势**: 空间数据基础设施全（地图+地理编码+行政区边界一家解决），Android SDK 原生支持
- **劣势**: 天气数据维度极少，独立配额仅 5,000次/月
- **推荐**: **空间数据基础设施 + 地图渲染引擎**，天气本身不做主力（应由和风负责）

### 3. 彩云天气 (caiyun) ⭐⭐⭐⭐
**已有适配器**: `src/lib/api/caiyun.ts`
- **免费版**: 1000次/天（注册认证后）
- **强项**: **分钟级降水预报**（Precipitation Nowcast），2小时内精准到街道
- **数据**: 实时+小时+天+分钟级降水+AQI
- **优势**: 降水预报国内最精准、API累计调用超1000亿次
- **推荐**: **降水专用源**，RainPulse核心数据

### 4. 心知天气 (seniverse) ⭐⭐⭐
**已有适配器**: `src/lib/api/seniverse.ts`
- **免费版**: 有限额度（具体未公开，估计400-1000次/天）
- **数据**: 实时/3天预报/生活指数
- **优势**: 轻量、易接入
- **劣势**: 免费版数据维度少，不如和风全面
- **推荐**: **轻量备选**，作为和风的对标验证

### 5. Open-Meteo ⭐⭐⭐⭐⭐ ✅ 调研完成！
**无适配器** — 建议立即可写适配器（零阻塞，无需Key）
- **免费版**: 300,000次/月（10,000次/天），非商业完全免费，无需 API Key
- **数据**: 全球模型(多个) + **CMA GRAPES 9km 中国专用模型**（best_match 自动选）
- **独家变量**: **体感温度**(apparent_temperature) — 和风/彩云/高德都没有！
- **降水概率**: 30次集合模拟 >0.1mm 概率，非主观估计
- **降水三分项**: rain(大尺度雨) / showers(对流降水) / snowfall(雪)
- **更多API族**: AQI(全球/欧洲)、季节性(7月)、气候(1950-2050)、海洋、洪水、卫星辐射
- **接口**: `https://api.open-meteo.com/v1/forecast?latitude=43.88&longitude=125.39&current=temperature_2m,relative_humidity_2m,wind_speed_10m,apparent_temperature,weather_code&hourly=temperature_2m,precipitation_probability,rain,showers&daily=temperature_2m_max,temperature_2m_min,precipitation_sum,sunrise,sunset&timezone=Asia/Shanghai`
- **优势**: 零成本、零注册、CORS支持、全球覆盖、体感温度+CMA中国模型
- **劣势**: AQI仅全球45km欧洲CAMS模型，中国无本地AQI；无天气预警；无分钟降水
- **推荐**: **温度/降水校验源**，提供体感温度+降水概率+CMA中国本地模型，与和风双引擎校验
